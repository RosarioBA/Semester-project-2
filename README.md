# BidLeaf Auction Website üçÉ
BidLeaf is an auction platform where users can list items for auction and bid on others' listings. Built with vanilla JavaScript and Tailwind CSS, it provides a clean, intuitive interface for auction interactions.

# Features üåü

## User Authentication

- Register with a stud.noroff.no email
- Login/logout functionality
- Update avatar and user information
- 1000 starting credits for new users

## Auction Listings

- View all active listings
- Search through listings (even when not logged in)
- Create new listings with title, description, and image
- Set auction deadlines
- View listing details and bid history

## Bidding System

- Place bids on active listings
- Track your active bids
- View won auctions
- Automatic credit management system

# Getting Started
**Prerequisites**

- Node.js (version 14 or higher)
- npm
- Git

## Installation

1. Clone the repository
bashCopygit clone https://github.com/yourusername/bidleaf.git

2. Install NPM packages
bashCopynpm install

3. Create a .env file in the root directory and add your API key:
envCopyVITE_API_KEY=your-api-key

4. Start the development server
bashCopynpm run dev
